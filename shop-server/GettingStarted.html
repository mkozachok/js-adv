Store API Reference

ALL requests require a key parameter, eg `localhost:3333/api/products?key=123`

Endpoint Method Example Sample Response Description
/api/products GET http://localhost:3333/api/products 
[
  { 
    "id": "1",
    "title": "iPhone X",
    "description": "Some text about iphone etc",
    "price": "1200",
    "image_src": "/images/iphone-x.png"
  },
  { 
    "id": "2",
    "title": "iPhone 8",
    "description": "Some text about iphone blah",
    "price": "800",
    "image_src": "/images/iphone-8.png"
  }
]
Дістає всі збережені продукти.
/api/products POST http://localhost:3333/api/products 
  {
    "id": "1",
    "title": "iPhone X",
    "description": "Some text about iphone etc",
    "price": "1200",
    "image_src": "/images/iphone-x.png"
  }
Зберігає новий продукт. Повертає збережений обєкт з id.
/api/products/:id GET http://localhost:3333/api/products/5 
  { 
    "id": "1",
    "title": "iPhone X",
    "description": "Some text about iphone etc",
    "price": "1200",
    "image_src": "/images/iphone-x.png"
  }
Дістає один запис за індифікатором.
/api/products/:id DELETE http://localhost:3333/api/products/5 
  {
    "status": "done"
  }
Видаляє продукт за індифікатором. Повертає статус повідомлення.
